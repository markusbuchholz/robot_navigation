# Open Robot Navigation


## Build

```bash

cd docker

sudo ./build.sh

```

## Run

```bash

cd docker

sudo ./run.sh

```


## Run example

Check [this] (https://docs.nav2.org/getting_started/index.html#running-the-example)

```bash

soure install/setup.bash

ros2 launch nav2_bringup tb3_simulation_launch.py headless:=False

```
